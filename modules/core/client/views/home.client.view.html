Who are we?

issle (Incorporating Student Standards Into Learning Engineering) is an application created to help teachers emphasize STEM in the classroom. By incorporating necessary standards into engineering tasks, students can learn engineering concepts while following State Common Core Standards.

Identify Common Core Standards

Discover standards that can be related to STEM activities in the classroom.

Create Engineering Projects With Connections to Common Core Curriculum

Utilize state standards in your engineering projects by referencing applicable objectives.

Review Projects Made by Other Teachers

Feel free to review existing projects made by other teachers and explore new ways to apply STEM in the classroom.
